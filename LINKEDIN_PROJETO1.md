# Texto revisado para LinkedIn — Projeto 1

## Versão sugerida (corrigida e objetiva)

**Projeto 1: Análise das Despesas do Governo Federal por Ministério**

Desenvolvi meu primeiro projeto completo de Data Analytics com foco em gastos públicos federais, utilizando dados do Portal da Transparência.

O objetivo foi analisar a evolução e a distribuição orçamentária entre os ministérios da **Educação, Saúde, Segurança e Defesa**.

Implementei um pipeline de ETL em Python com tratamento de dados e carga em SQL Server. Em seguida, construí um dashboard no Power BI para facilitar a visualização dos principais insights.

💡 **Principais entregas**
- Pipeline de ETL em Python com limpeza e padronização dos dados
- Modelagem relacional e carga em SQL Server para análise
- Dashboard em Power BI com comparativos por ministério e evolução temporal

🧰 **Tecnologias utilizadas**
Python · pandas · SQL Server · Power BI · ETL · GitHub
